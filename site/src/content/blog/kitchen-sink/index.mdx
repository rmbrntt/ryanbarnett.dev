## Callout Components - Academic/Mathematical

<Callout variant="definition" title="Continuous Function">
  A function $f: X \to Y$ between topological spaces is **continuous** if for every open set $V \subseteq Y$, the preimage $f^{-1}(V)$ is open in $X$.
  
  For metric spaces, this is equivalent to the $\epsilon$-$\delta$ definition.
</Callout>

<Callout variant="theorem" title="Fundamental Theorem of Calculus">
  Let $f$ be a continuous real-valued function defined on a closed interval $[a, b]$. If $F$ is defined by
  
  $$F(x) = \int_a^x f(t)\,dt$$
  
  then $F$ is uniformly continuous on $[a, b]$ and differentiable on $(a, b)$, and $F'(x) = f(x)$.
  
  <Callout variant="proof">
    Let $x, x + h \in (a, b)$. Then:
    
    $$\frac{F(x+h) - F(x)}{h} = \frac{1}{h}\int_x^{x+h} f(t)\,dt$$
    
    By the mean value theorem for integrals, there exists $c \in [x, x+h]$ such that:
    
    $$\frac{1}{h}\int_x^{x+h} f(t)\,dt = f(c)$$
    
    As $h \to 0$, we have $c \to x$, and by continuity of $f$, $f(c) \to f(x)$.
    
    Therefore, $F'(x) = f(x)$. □
  </Callout>
</Callout>

<Callout variant="lemma" title="Zorn's Lemma">
  Every non-empty partially ordered set in which every chain has an upper bound contains at least one maximal element.
  
  <Callout variant="remark">
    Zorn's Lemma is equivalent to the Axiom of Choice and the Well-Ordering Theorem.
  </Callout>
</Callout>

<Callout variant="corollary" title="Consequence of the Mean Value Theorem">
  If $f'(x) = 0$ for all $x$ in an interval $(a, b)$, then $f$ is constant on $(a, b)$.
  
  <Callout variant="intuition">
    If the derivative is always zero, the function has no rate of change, hence it must be flat (constant).
  </Callout>
</Callout>

<Callout variant="proposition">
  Every finite dimensional vector space over a field $F$ is isomorphic to $F^n$ for some $n \in \mathbb{N}$.
</Callout>

<Callout variant="axiom" title="Axiom of Extensionality">
  Two sets are equal if and only if they have the same elements:
  
  $$\forall A \forall B [\forall x (x \in A \iff x \in B) \implies A = B]$$
</Callout>

<Callout variant="conjecture" title="Goldbach's Conjecture">
  Every even integer greater than 2 can be expressed as the sum of two primes.
  
  <Callout variant="example">
    - $4 = 2 + 2$
    - $6 = 3 + 3$
    - $8 = 3 + 5$
    - $10 = 3 + 7 = 5 + 5$
  </Callout>
</Callout>

<Callout variant="problem" title="Challenge Problem">
  Given a sorted array of integers, implement a function that finds all pairs of numbers that sum to a target value. Your solution should have O(n) time complexity.
  
  <Callout variant="solution" defaultOpen={false}>
    Use the two-pointer technique:
    
    ```python
    def find_pairs(arr, target):
        pairs = []
        left, right = 0, len(arr) - 1
        
        while left < right:
            current_sum = arr[left] + arr[right]
            if current_sum == target:
                pairs.append((arr[left], arr[right]))
                left += 1
                right -= 1
            elif current_sum < target:
                left += 1
            else:
                right -= 1
        
        return pairs
    ```
    
    Time complexity: O(n), Space complexity: O(1) excluding output
  </Callout>
</Callout>
